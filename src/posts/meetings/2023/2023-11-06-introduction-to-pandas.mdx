This workshop will introduce participants to the basics of data analysis
using the [pandas](https://pandas.pydata.org/) library in the Python programming language. Pandas is a
foundational library for modern data science in science and industry. We
will use pandas to load, process, and transform tabular and timeseries
data. We will then visualize it with [Matplotlib](https://matplotlib.org/), the Python plotting
library. No prior programming experience is required.
